# Components

## DymikForm

The `DymikForm` component is a reusable form component provided by `dymik-core`. It is designed to simplify form creation and validation in Vue.js applications.

### Example

```vue
<template>
  <DymikForm />
</template>

<script setup>
import { DymikForm } from 'dymik-core';
</script>
```
